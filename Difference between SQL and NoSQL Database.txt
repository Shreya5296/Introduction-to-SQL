SQL databases are table based.Different tables are created for different data.
SQL databases strictly rely on relations. This means that that data is present in separate tables with a relation between them.
SQL databases have structured predefined schemas. All the columns in a particular table define the data type of all the data.
SQL databases are better with multi row based structured data.
